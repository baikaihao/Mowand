## English

Mowand is a macOS mouse gesture automation app. Draw directional gestures with your mouse, like waving a magic wand, and bind them to system actions, keyboard shortcuts, app launches, scripts, and automation workflows.

### Features

- Eight-direction gesture recognition.
- Global mouse gesture support.
- Built-in default gesture templates.
- Gesture list and gesture editor.
- Action chains for running multiple steps in order.
- HUD feedback while drawing gestures.
- App-specific rule configuration foundation.
- Accessibility permission guidance.
- General settings for launch behavior and Dock visibility.
- Native macOS app icon and settings-page icon.

### Default Gestures

| Gesture | Default Action |
|---|---|
| Left | Back |
| Right | Forward |
| Down | Volume Down |
| Up | Volume Up |
| Right -> Down | Refresh |
| Down -> Right | Screenshot |
| Up -> Down | Show Desktop |
| Left -> Right | Switch to Recent App |
| Up Right | Brightness Up |
| Down Right | Brightness Down |
| Down Left | Mute |
| Up Left | Lock Screen |

### Permissions

Mowand requires macOS Accessibility permission for global mouse monitoring and simulated input. Some settings, especially Dock icon visibility, may be most reliable after restarting the app.

### Status

Mowand v1.0.0 is the first public release.

## 中文

Mowand（魔杖）是一款 macOS 鼠标手势自动化工具。你可以像挥动魔杖一样绘制方向手势，并将手势绑定到系统操作、快捷键、App 启动、脚本和自动化流程。

### 功能

- 八方向手势识别。
- 全局鼠标手势支持。
- 内置默认手势模板。
- 手势列表和手势编辑器。
- 支持按顺序执行多个步骤的动作链。
- 绘制手势时提供 HUD 反馈。
- 提供应用专属规则配置基础。
- 提供辅助功能权限引导。
- 提供启动行为、Dock 图标显示等通用设置。
- 内置 macOS App 图标和设置页图标。

### 默认手势

| 手势 | 默认动作 |
|---|---|
| 左 | 返回 |
| 右 | 前进 |
| 下 | 降低音量 |
| 上 | 提高音量 |
| 右 -> 下 | 刷新 |
| 下 -> 右 | 截图 |
| 上 -> 下 | 显示桌面 |
| 左 -> 右 | 切换最近使用的 App |
| 右上 | 提高亮度 |
| 右下 | 降低亮度 |
| 左下 | 静音 |
| 左上 | 锁屏 |

### 权限

Mowand 需要 macOS 辅助功能权限，才能进行全局鼠标监听和模拟输入。部分设置，尤其是 Dock 图标显示相关设置，在重启 App 后最稳定。

### 状态

Mowand v1.0.0 是首个公开版本。
